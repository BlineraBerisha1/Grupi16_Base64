# Grupi16_Base64
Base64 Encoder and Decoder-Data Security

This project is a simple Base64 Encoder and Decoder developed for the Data Security class.
We used it to understand how data can be converted into Base64 format and then decoded back to its original form.

⚙️ Features
Encode text to Base64
Decode Base64 to text
Easy to use

🛠️ Technology
Java

🔐 Note
Base64 is not encryption, it is only used for encoding data.
